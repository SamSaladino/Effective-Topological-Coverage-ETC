# ETC – Effective Topological Coverage

A computational framework to assess how well observed nodes (e.g. metabolites) cover the topology of a biological network using Laplacian diffusion and information-theoretic principles.

## Features
- Propagated signal smoothing using Laplacian inversion
- ETC score quantifying network coverage
- Entropy of coverage spread
- Visualization utilities for graph heatmaps
